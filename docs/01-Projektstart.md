# Projektstart

## Ziel des Projekts
Im Rahmen dieser ESA entwickle ich eine verteilte Anwendung zur Verwaltung von Fahrzeugen und Wartungen. Ziel ist es die Anwendung schrittweise mit Unterstützung von KI zu entwickeln und dabei jeden Entwicklungsschritt nachvollziehbar zu dokumentieren.

## Projektstruktur
Zu Beginn habe ich die Projektstruktur angelegt und das Projekt in mehrere Module aufgeteilt:
- backend
- reminder-service
- notification-service
- docs

Durch diese Aufteilung kann jedes Modul später eine eigene Aufgabe übernehmen und unabhängig entwickelt werden, so wie es in in der ESA gefordert ist.

## Entwicklungsumgebung
Für die Entwicklung verwende ich:
- Visual Studio Code
- Git
- GitHub
- Node.js
- Gemini CLI

Die Entwicklung erfolgt in kleinen Schritten step by step. Nach jedem wichtigen Schritt wird der aktuelle Stand dokumentiert und im Git Repository gespeichert.
